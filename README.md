# tryskills.sh

**One URL prefix to instantly try any agent skill.**

```
skills.sh/anthropics/skills/frontend-design       ← browse
tryskills.sh/anthropics/skills/frontend-design     ← try it now
```

> Add `try` before `skills.sh` — get a live agent session with that skill loaded.

## How It Works

1. Find any skill on [skills.sh](https://skills.sh) or GitHub
2. Change the URL prefix to `tryskills.sh`
3. Enter your LLM API key + Daytona sandbox key
4. Click **Launch** — chat with an agent running that skill

All compute runs on **your own API keys**. tryskills.sh is a static web app with zero server costs.
